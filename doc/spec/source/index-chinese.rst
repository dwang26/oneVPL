.. _VPL-section:

================
|vpl_short_name|
================

|vpl_full_name| (|vpl_short_name|) 是一个用于视频解码、编码和处理的编程接口，
用于在 CPU、GPU 和其他加速器上构建可移植的视频管道。它提供视频处理和视频分析工作负载中的设备发现和选择的API，以及用于零拷贝缓冲区共享的 API 原语。|vpl_short_name| 向后兼容且跨架构兼容，以确保在当前和下一代硬件上实现最佳执行，而无需更改源代码。

.. rubric:: 规范的版本

本文档包含 |vpl_short_name| 规范的版本 |vpl_version|。

|vpl_short_name| 规范的最新发布版本: `<https://intel.github.io/libvpl>`__.
